# VS Code Cloud Terminal

24/7 VS Code terminal running on Render Free Tier with Uptime Robot.

## Features
- VS Code in browser
- 24/7 uptime with Uptime Robot
- Colab automation for Minecraft
- Auto-reconnect and monitoring

## Setup
1. Deploy to Render
2. Configure Uptime Robot
3. Access VS Code at: `https://YOUR_APP.onrender.com:8080`

## Default Credentials
- Password: `coder123` (change in config.yaml)

## Automation
- Minecraft Colab automation auto-starts
- Manual login required once: `python3 scripts/manual_login.py`
